# Security Policy

## Supported Versions

Security updates are provided for the latest public release of Local Agent Studio.

| Version | Supported |
| ------- | --------- |
| 0.2.x   | :white_check_mark: |
| 0.1.x   | :x: |
| < 0.1   | :x: |

## Reporting a Vulnerability

If you find a security vulnerability in Local Agent Studio, please report it privately instead of opening a public issue.

You can report vulnerabilities through GitHub Security Advisories:

https://github.com/CrazyDashTool/Local-Agent-Studio/security/advisories/new

Please include:

- A clear description of the issue
- Steps to reproduce it
- The affected version
- Your operating system
- Any logs, screenshots, or proof-of-concept details if available

I will try to respond as soon as possible. If the issue is accepted, I will work on a fix and publish a security update. If the report is declined, I will explain why.

Please do not publicly disclose the vulnerability until a fix is available.
