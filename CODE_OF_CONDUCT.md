# Code of Conduct

## Our Pledge

We want Local Agent Studio to be a welcoming and useful project for everyone.
Everyone participating in this project is expected to be respectful, constructive,
and focused on helping the project improve.

## Expected Behavior

- Be respectful of different experience levels and backgrounds.
- Give clear, constructive feedback.
- Keep discussions focused on the project.
- Assume good intent, but accept corrections when something causes harm.
- Help make issues, discussions, and pull requests easier to understand.

## Unacceptable Behavior

- Harassment, insults, threats, or personal attacks.
- Discriminatory language or behavior.
- Spam, trolling, or intentionally disruptive comments.
- Publishing someone else's private information.
- Repeatedly derailing technical discussions.

## Enforcement

Project maintainers may edit, hide, or remove comments, issues, pull requests,
or other contributions that do not follow this Code of Conduct.

Maintainers may also temporarily or permanently block users from participating
in the project if needed to keep the community healthy.

## Reporting

If you see behavior that violates this Code of Conduct, please contact the
project maintainer privately or use GitHub's reporting tools.

Reports will be reviewed as soon as possible. The maintainer will decide what
action is appropriate based on the context and severity of the situation.
